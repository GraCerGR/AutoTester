## <br> Команды
### Запуск проекта
Проект запускется командой, находясь при этом в корне проекта.  
```
go run .  
```

### Дополнительные команды 
#### parsexml
Команда `parsexml` предназначена для парсинга результатов автотестов (.xml) в понятный чекеру формат (.json).
```
go run . parsexml <folder> <index_bool> 
```
<sub>folder - папка с вашими файлами .xml;</sub>  
<sub>index_bool - необязательная переменная, устанавливает в конце каждого файла индекс.</sub>

## Настройка
Для натройки конфигурации системы используются файлы в папке settings:  
- stack.go - настраивает пути к образам для каждого языка программирования.  
- config.go - управляет настройками контейнеров: их количество, папки и т. д.
